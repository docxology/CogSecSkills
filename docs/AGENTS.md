# AGENTS.md - Public Documentation

This directory is public reader documentation for the skill system.

## Source And Generated Boundaries

- quality-dashboard.md is generated by `python -m cogsecskills dashboard --write`.
  Do not edit it by hand.
- skill-worked-examples.md is generated by `python -m cogsecskills examples --write`.
  Do not edit it by hand.
- `ageint/` is the educational upstream surface; keep AGEINT claims separate
  from CogSecSkills local conformance claims.
- Command references must match `src/cogsecskills/cli.py`.
- Optional harness documentation must use the three status labels: default adapters,
  configured structural adapters, and documented external profiles.

## Claim Rules

- Local gates prove source synchronization, structural multiharness conformance,
  deterministic scenario fixture readiness, deterministic worked-example
  coverage, and render readiness.
- They do not prove live external runtime behavior, connector safety, field
  effectiveness, public release status, peer review, vendor endorsement, or DOI
  archival.
- Keep per-skill `refs` metadata distinct from scholarly citations in
  `manuscript/references.bib`.
